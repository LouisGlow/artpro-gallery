ArtPro Gallery — Sunninghill Village photos
===========================================

To show this gallery's photos on the website:

1. Put photos in THIS folder, named 01.jpg, 02.jpg, 03.jpg … in order.
2. That's it. The gallery page picks them up automatically — no code changes.

Notes
-----
- Use .jpg files, two-digit names: 01.jpg, 02.jpg … (up to 80).
- 01.jpg is shown as the big hero image — make it your best/storefront shot.
- Keep each file a few hundred KB to ~1 MB.
- The gallery page: gallery.html?g=sunninghill-village
